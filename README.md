## Usage

### Common keybindings

| Key               | Description  |
| ----------------- | ------------ |
| <kbd>j</kbd>      | cursor down  |
| <kbd>k</kbd>      | cursor up    |
| <kbd>Tab</kbd>    | toggle focus |
| <kbd>Ctrl+C</kbd> | quit         |
